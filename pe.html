파트별 교육 세션


PM

  • 1주차: PM의 정의와 기획하는 과정 전반

  • 2주차: 프로젝트 시작 및 계획, 그리고 생애주기

  • 3주차: 플로우 차트와 와이어프레임으로 소통하기

  • 4주차: 개발 프로젝트 회의 관리 & PM - DE 협업 프로젝트

  • 5~6주차: PM - DE 협업 프로젝트

DESIGN

  • 1~2주차: UX & UI의 이해, 다른 파트와의 협업 방식

  • 3주차: Figma의 이해(리소스, 컴포넌트, 오토레이아웃, 플러그인과 템플릿 사용법)

  • 4주차: Figma의 심화(브랜딩과 로고 디자인, 와이어프레임, 무드보드 만들기) & PM - DE 협업 프로젝트

  • 5~6주차: PM - DE 협업 프로젝트

FE

  • 1~3주차: 실전에서 써먹는 JavaScript

  • 4주차: CSS 고급 스타일링

  • 5~6주차: 개발부터 배포까지, 나만의 일정 관리 앱

BE

  • 1주차: 웹과 Django 프레임워크 기초 익히기

  • 2주차: DRF 기초와 API 이해하기

  • 3~4주차: DRF를 이용해 영화 리뷰 서비스 API 구축하기

  • 5~6주차: 나만의 일정 관리 서비스 API 구축 및 배포
