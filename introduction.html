멋쟁이사자처럼 대학 소개


멋쟁이사자처럼은 누구나 프로그래밍을 통해 자신만의 웹서비스를 구현하고, 꿈을 이루기 위한 프로그래밍 교육 단체입니다.


멋쟁이사자처럼 대학은 테크 기반의 아이디어 실현을 위한 전국 최대 규모의 대학 연합 창업 동아리입니다.

이번 11기에서는 기획, 디자인, 프론트엔드, 백엔드 트랙별로 진행하며, 시그니처 프로그램인 아이디어톤 / 해커톤 / 장기 프로젝트가 예정되어 있습니다.


  1. 온/오프라인 학습

    • 멋사의 온라인 강의 플랫폼을 통해 초급자를 위한 다양한 VOD 강의를 무료로 제공합니다.

    • 공통 교육 세션 / 파트별 교육 세션을 통해 웹 개발을 직접 배웁니다.


  2. 중앙 아이디어톤

    • 팀 프로젝트를 통해 실현하고 싶은 내 아이디어(웹 서비스)를 발표하는 자리입니다.

    • 전국의 멋대 학생들이 모여 아이디어를 공유하는 네트워킹의 장입니다.


  3. 서비스 빌딩을 위한 입체적 교육

    • 멋대에서는 웹 개발 교육(HTML / CSS / JS, Python / Django) 뿐만 아니라, 서비스 기획, UI / UX 디자인 등 자신의 서비스를 빌딩하기 위한 다양한 역량을 배울 수 있습니다.


  4. 해커톤

    • 실제 아이디어가 구현되는 축제의 장입니다. 밤샘 코딩 및 최종 발표를 합니다.

    • 창업까지 연계되는 사례가 다수 있으며, 서비스가 태어나는 과정을 직접 경험할 수 있게 됩니다.
